# Content Studio Basics

Cardbey Content Studio allows you to design images for screens, stores, ads, and social content.

## What You Can Create

- Posters
- Menu screens
- Product cards
- Branding layouts
- Social media posts
- Screen content for playlists

## Key Features

### Canvas Editor
- Drag and drop images or text
- Resize, rotate, duplicate elements
- Background color or image
- Layer controls (bring forward, send back)

### AI Tools
- Auto-generate layouts
- Rewrite text snippets
- Suggest color palettes
- Create menu screens from a photo (OCR → layout)

### Asset Library
- Upload images or videos
- Insert assets directly into your design
- Recently used assets appear automatically

## Creating a New Design

1. Click **New Design**.
2. Choose a template or blank canvas.
3. Add elements:
   - Text
   - Images
   - Shapes
4. Adjust layout using the side panel:
   - Size
   - Colors
   - Fonts
   - Alignment

## Exporting or Publishing

You can:
- Export the design as PNG or JPG.
- Add it to a playlist.
- Save it in the asset library.
- Send it directly to a screen.

## Troubleshooting

### Canvas looks stretched or clipped
- Check that you selected the correct canvas size (16:9, 9:16).

### AI tools not responding
- Try refreshing the page.
- Check network connection.
- Ensure the backend AI service is running.

### Uploaded image not appearing
- Ensure the image meets file requirements.
- Try re-uploading or checking console errors.
