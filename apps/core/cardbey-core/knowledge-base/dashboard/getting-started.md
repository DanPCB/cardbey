# Getting Started with the Cardbey Dashboard

The Cardbey Dashboard is the main control center for managing stores, devices, playlists, and content.

## What You Can Do in the Dashboard

- Create a store (food, product, service, listing)
- Add items: products, services, menus
- Manage playlists for screens
- Pair and configure devices (tablets, TVs)
- Access Content Studio for design work

## First-Time Setup

1. Log into the dashboard.
2. If you do not have a store yet:
   - Click **Create Store**
   - Choose the store type (Food, Product, Service, etc.)
   - Enter name, logo, and basic info.

3. Once a store exists:
   - You can start adding items or assign screens.

## Creating Products or Services

1. Go to your **Store**.
2. Click **Add Item**.
3. Choose:
   - Product
   - Service
4. Enter:
   - Name
   - Description
   - Price
   - Category
   - Images

Items will appear instantly on your public store page.

## Managing Devices

1. Go to **Devices**.
2. Click **Pair New Device**.
3. Enter the code shown on the tablet/TV.
4. Once paired, you can:
   - Assign playlists
   - Change orientation
   - View status
   - Rename or re-locate the device

## Managing Playlists

1. Go to **Playlists**.
2. Create a new playlist or edit an existing one.
3. Add images or videos.
4. Assign to one or more devices.

## Troubleshooting

### Can't access dashboard?
- Ensure you're logged in.
- Try refreshing or clearing browser cache.

### Device not appearing?
- Confirm pairing code was entered correctly.
- Device must be online during pairing.

### Changes not showing on device?
- Device may be caching data.
- Wait for heartbeat (usually within a few seconds).
- Restart device app if necessary.
